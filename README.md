# Getting started

### Play game

<a href="https://starpandax.itch.io/blockpath">Play Online</a> WebGL version

### Build

1. Clone the repo
    ```bash
    git clone https://github.com/StarPandaBeg/BlockPath.git
    ```
2. Open Project in **Unity 2022.3.13f1** or later
3. Configure & Build project
